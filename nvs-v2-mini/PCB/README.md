## Connection

| ESP Pin | Function    | Note               |
| ------- | ----------- | ------------------ |
| 5       | Relay       | active low         |
| 4       | IR          | control humidifier |
| 13      | Door sensor | passive            |
| 12      | PIR         | active high        |
